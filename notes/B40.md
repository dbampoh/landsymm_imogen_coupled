## 2. Source-level confirmation (the audit findings)

### 2.1 The smoking gun — N-deposition added to soil mineral N pool without source-discrimination

`lpjguess/modules/somdynam.cpp:1689-1691`:

```cpp
// Monthly nitrogen addition to the system
soil.nmass_inc((gridcell.aNH4dep + soil.anfix_mean) / 12.0, NH4);
soil.nmass_inc(gridcell.aNO3dep / 12.0, NO3);
```

**Mechanism**:
- `gridcell.aNH4dep` = annual atmospheric NHx (ammonia + ammonium) deposition (kgN/m²/yr); populated from the input module's ndep NetCDFs (v1.0: `imogencfx::getclimate_for_gridcell` → `imogencfx.cpp:1281-1282 + 1740-1741` per the `cfx`/`imogencfx` ndep ingestion pathway)
- `gridcell.aNO3dep` = annual atmospheric NOy (NOx + nitrate) deposition (kgN/m²/yr); same ingestion path
- `soil.anfix_mean` = annual biological N-fixation rate (kgN/m²/yr); computed at `somdynam.cpp:1511` from a precipitation-driven empirical formula
- The 3-source N input is partitioned monthly: NH4 pool gets (NHx-dep + N-fix)/12 each month; NO3 pool gets NOy-dep/12 each month
- **No source-discrimination**: the resulting `soil.NH4_mass + NO3_mass` arrays carry summed contributions from all sources without per-source accounting

### 2.2 Nitrification + denitrification process the combined pool

`lpjguess/modules/ntransform.cpp` (569 LOC; cited papers: Xu-Ri 2008, Parton 1996, Ma et al. 2022 JAMES):

| Line | Function | What it does |
|---|---|---|
| 179 | `nitrification(Patch&, Soil&, Climate&)` | Daily nitrification + co-production of NO + N2O gases (per Xu-Ri 2008 table 8 eqn 1-2 + Parton 1996 pH limiting factor) |
| 230 | `n2o_inc = ngas_inc - no_inc` | NO + N2O byproducts split per fraction `f_no` from soil moisture/temp |
| 232 | `soil.N2O_mass_d += n2o_inc` | **N2O increment added to soil N2O pool from nitrification** |
| 275 | `denitrification(Patch&, Soil&, Climate&)` | Daily denitrification (NO3 → NO2 → NO → N2O → N2 cascade; per Xu-Ri 2008 table 9 + Ma 2022 JAMES kinetics) |
| 367-368 | `n2o_inc = ngas_inc * f_n2o_n2_w * f_n2o_n2_T * f_n2o_n2_pH` | N2O fraction of denitrification gas-product |
| 373 | `soil.N2O_mass_w += n2o_inc` | **N2O increment added from denitrification** |
| 467 | `patch.fluxes.report_flux(Fluxes::N2O_SOIL, n2o_d_flux_inc + n2o_w_flux_inc)` | **THE AGGREGATE OUTPUT**: dry-soil + wet-soil N2O increments reported as a SINGLE channel `Fluxes::N2O_SOIL` (no per-source attribution; no separate "natural" vs "N-dep-influenced" channels) |

**Critical observation**: the nitrification + denitrification kinetics depend on the soil mineral N pool's MAGNITUDE (NH4_mass + NO3_mass), which has been incremented by ALL 3 N inputs (atmospheric NHx-dep + atmospheric NOy-dep + biological N-fix). A larger N pool → faster nitrification + denitrification → more N2O production. **There is no mechanism in the LPJG architecture to tag N2O production with its upstream N source.**

### 2.3 The output channel pathway — single aggregate N2O channel

patch.fluxes.get_annual_flux(Fluxes::N2O_FIRE) * to_gc_avg;   // line 221

So the IMOGEN-handshake natural-N2O channel (`imogen_lpjg_ch4_n2o_flux.txt` column 3) = sum of N2O_FIRE + N2O_SOIL fluxes, multiplied by 44/28 to convert from kgN/yr → kgN2O/yr → aggregated to global TgN2O/yr.

**No source-discrimination preserved through the output pipeline.** The same single channel carries:
- "Purely natural" N2O from native soil mineral N + organic-matter mineralization
- "N-deposition-influenced natural" N2O from atmospheric NHx + NOy deposition perturbing soil mineral N pool
- Biological N-fixation-derived N2O

The "natural" attribution flows from the channel-naming convention (`N2O_SOIL` excluding fertilizer-direct N2O which is handled separately via cropland management code paths) — not from per-source accounting within the process kinetics.

## 3. The mechanism — quantitative narrative

### 3.1 Pre-industrial baseline (1900)

- Atmospheric NHx + NOy deposition globally: ~4-8 TgN/yr (pre-industrial baseline per Lamarque et al. 2010 + Galloway et al. 2004 reconstructions)
- LPJG's `gridcell.aNH4dep + aNO3dep` (post-cfx-ingestion of pre-industrial-era ndep NetCDF): low values matching this baseline
- Soil mineral N pool: minimally perturbed beyond the biological-N-fixation + native-mineralization baseline
- Nitrification + denitrification: at "natural" baseline rates per Xu-Ri 2008 kinetics
- LPJG output N2O_SOIL: ~10-11 TgN2O/yr (per B20 1900 = 10.76 TgN2O/yr; close to Tian 2020 envelope mean 9.6 TgN2O/yr; within envelope)

### 3.2 Modern peak (2000-2017)

- Atmospheric NHx + NOy deposition globally: ~60-65 TgN/yr (industrial-era peak per Lamarque et al. 2010 + Hauglustaine et al. 2014; 10-15× pre-industrial)
- LPJG's `gridcell.aNH4dep + aNO3dep` (post-cfx-ingestion of modern-era ndep NetCDF): elevated proportionally
- Soil mineral N pool: substantially perturbed; mean NH4_mass + NO3_mass elevated by ~30-50% over pre-industrial in well-deposited regions (Europe, eastern US, southern China per ndep spatial patterns)
- Nitrification + denitrification: accelerated due to elevated N pool magnitude (Xu-Ri 2008 kinetics scale roughly linearly with soil N availability up to saturation; non-linear above)
- LPJG output N2O_SOIL: ~14 TgN2O/yr (per B20 2008-2017 = 14.43 TgN2O/yr; ~50% above Tian 2020 envelope max 11.5 TgN2O/yr; CONSISTENT WITH Saikawa 2014's wider envelope ~9-15 TgN/yr × 44/28 = ~14-24 TgN2O/yr modern-decade total natural)

### 3.3 SSP1-2.6 mid-century decline (2050-2100)

- SSP1-2.6 is the strong-mitigation scenario; anthropogenic NHx + NOy emissions decline from peak 2020 ~60 TgN/yr → ~30-40 TgN/yr by 2050 (per Bauer et al. 2017 + Rao et al. 2017 SSP1-2.6 reactive-nitrogen trajectories)
- LPJG's `gridcell.aNH4dep + aNO3dep` follows ndep NetCDF trajectory (declining)
- Soil mineral N pool: returns toward pre-industrial baseline (asymptotically; modulated by soil-N residence times)
- Nitrification + denitrification: decelerated
- LPJG output N2O_SOIL: ~9.6 TgN2O/yr by 2050 (per B20 2050 = 9.57 TgN2O/yr; basically matches Tian 2020 envelope mean 9.6 TgN2O/yr; tightly within envelope)
- LPJG output N2O_SOIL: ~9.9 TgN2O/yr by 2090 (per B20 2090 = 9.94 TgN2O/yr; within envelope)

### 3.4 Full-envelope time-mean

- LPJG output N2O_SOIL: 11.18 TgN2O/yr (per B20 full_envelope 1900-2100 = 11.18 TgN2O/yr)
- Tian 2020 envelope: 9.6 [8.3-11.5] TgN2O/yr
- Δ_rel: +16.5% above mean; WITHIN envelope max
- Verdict: ✅ WITHIN_ENVELOPE_MEAN_WITH_TIME_VARIATION (per B20 overall verdict)

## 4. Tian 2020 vs Saikawa 2014 envelope-definition reconciliation

The seeming mismatch between the modern-decade hump (14.43 TgN2O/yr) and Tian 2020 envelope (max 11.5 TgN2O/yr) is **a definitional difference, not a model-vs-observation disagreement**:

| Reference | Envelope (modern decade; TgN2O/yr) | Definition of "natural N2O" | LPJG-N2O_SOIL alignment |
|---|---|---|---|
| **Tian 2020 Nature** | 9.6 [8.3-11.5] | "Natural soil flux" = pre-industrial-equivalent baseline; conceptually EXCLUDES N-deposition-perturbed natural pathway | LPJG full-envelope mean (11.18) WITHIN envelope; modern-decade peak (14.43) ~25% above envelope max |
| **Saikawa 2014 ACP** | ~9-15 TgN/yr × 44/28 = ~14-24 TgN2O/yr (modern decade) | "Natural soil flux + N-deposition-influenced natural" wider envelope; conceptually INCLUDES N-deposition-perturbed natural pathway | LPJG modern-decade peak (14.43) at LOWER end of Saikawa envelope; full-envelope mean (11.18) within lower portion |

**Reconciliation**: LPJG's `N2O_SOIL` channel scope matches the Saikawa 2014 wider definition, not the Tian 2020 narrower definition. The full-envelope time-mean is the most-defensible single-number comparison (since it averages across the modern-N-perturbed era + pre-industrial + future-mitigated periods), and it passes both definitions' envelopes (within Tian 2020 [8.3-11.5]; comfortably within Saikawa 2014 [~14-24]). The decadal-time-resolved comparison shows the modern hump is well-explained by the N-deposition-influenced pathway.

**Paper §2.4.4 sector-ownership rule recommended addition**:

> _"LPJG's natural-soil N2O channel (the `Fluxes::N2O_SOIL` flux reported in `ngases.out` and propagated to the IMOGEN-handshake natural-N2O channel at `imogen_lpjg_ch4_n2o_flux.txt`) is computed via Xu-Ri 2008 + Ma et al. 2022 JAMES nitrification + denitrification kinetics applied to the soil mineral N pool, which is incremented monthly by the sum of atmospheric NHx + NOy deposition (from ndep NetCDF input) + biological N-fixation (per Cleveland et al. 1999 precipitation-driven empirical formula) + organic-matter mineralization (from native + decomposed litter). ## 5. Newly-surfaced audit item (NEW B46 filed at this commit; optional v1.5+ source-edit)

### B46 NEW — Optional v1.5+ source-edit: split LPJG's `N2O_SOIL` into separately-attributed channels

**Title**: Optional v1.5+ source-edit: emit separate `Fluxes::N2O_SOIL_NATURAL` + `Fluxes::N2O_SOIL_NDEP_INFLUENCED` channels in `ntransform.cpp` for per-source attribution accounting

**Trigger**: B40 source-reading at `lpjguess/modules/ntransform.cpp:179-467` + `somdynam.cpp:1689-1691` confirmed that the current single-channel `Fluxes::N2O_SOIL` aggregates N2O production from (a) "purely natural" N inputs (biological N-fixation + organic-matter mineralization + native soil mineral N) and (b) "N-deposition-influenced natural" N inputs (atmospheric NHx + NOy deposition perturbing soil mineral N pool). This single-channel design is consistent with the published Smith et al. 2014 / Olin et al. 2015 LPJ-GUESS architecture; B40 explanatory study confirms the modern-decade hump is well-explained by the N-deposition-influenced pathway. **Optional refinement**: emit separately-attributed channels so future v1.5+ scientific work can quantitatively partition the "natural" vs "N-deposition-influenced natural" components of LPJG's soil N2O output.

**Scope**: introduce new flux channels:
- `Fluxes::N2O_SOIL_NATURAL` — N2O from N-fixation + organic-matter mineralization + native soil mineral N
- `Fluxes::N2O_SOIL_NDEP_INFLUENCED` — N2O from atmospheric NHx + NOy deposition-perturbed soil mineral N pool

Implementation requires per-source attribution accounting within the nitrification + denitrification kinetics in `ntransform.cpp` — non-trivial because the kinetics operate on the AGGREGATE soil mineral N pool, not on per-source-tagged sub-pools. Options:
- **(α)** Tag the soil mineral N pool with source attribution (split `soil.NH4_mass` into `NH4_mass_natural + NH4_mass_ndep`; same for NO3_mass); track separate tendencies through nitrification/denitrification. ~150-250 LOC across `ntransform.cpp` + `soil.cpp` + `framework/guess.h`.
- **(β)** Compute attribution post-hoc via ratio: N2O_NATURAL = N2O_SOIL × (natural_N_input / total_N_input); N2O_NDEP_INFLUENCED = N2O_SOIL × (ndep_N_input / total_N_input). ~50-80 LOC; simpler but less accurate (linear approximation; ignores non-linear kinetics interactions).
- **(γ)** Defer entirely; rely on Saikawa-aligned interpretation per §4 of this writeup. ~0 LOC.

Plus output-side updates:
- `commonoutput.cpp` ngases.out columns: add `N2O_SOIL_NATURAL` + `N2O_SOIL_NDEP_INFLUENCED` rows (~10-15 LOC)
- `imogenoutput.cpp` IMOGEN-handshake channel: decide whether to keep aggregated single channel OR add separate channels for downstream analysis (~10-20 LOC)
- `framework/guess.h` Fluxes enum extension (~5 LOC)

**Effort**: 100-200 LOC if option (α) full; 60-90 LOC if option (β) ratio-based.

**Priority**: **LOW** — B40 explanatory study + paper §2.4.4 sector-ownership rule recommended addition (per §4 above) is sufficient for the v1.0 paper-publication narrative; the optional source-edit is purely analytical-refinement value for v1.5+ work where quantitative natural-vs-N-deposition-influenced partitioning would be useful (e.g., for sensitivity-experiment analyses; tighter cross-validation against Saikawa 2014 wider envelope partition; etc.).

**Acceptance test**: re-run B20 with the new channels; verify `N2O_SOIL_NATURAL` time-mean is closer to Tian 2020 envelope mean (9.6 TgN2O/yr) + `N2O_SOIL_NDEP_INFLUENCED` time-mean tracks the atmospheric NHx + NOy emissions trajectory shape; document the partition in an extended B20 validation report.

**Backport classification**: **TRUNK-RELEVANT** if option (α) full implementation (touches canonical `ntransform.cpp` + Fluxes enum in `framework/guess.h` — both shared with `trunk_r13078/` per `notes/TRUNK_R13078_BACKPORT_LEDGER.md` discipline).
