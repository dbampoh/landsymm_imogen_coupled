##### 3.4.2.2 Side-investigation — double-counting concern (raised by user 2026-05-16 ~20:36 UTC+2)

User raised a substantive scientific concern: the `intermediary_py` system consumes 18 `.gz` files at `intermediary_py/imogen_ghg_controller/inputs/lpjg/{historical/,scenarios/ssp[1-5]_rcp*/}` (~1.3 GB; cached LPJ-GUESS Stage-II output from a prior PLUM-v2-driven cluster run; same code as our rebuild's LPJG produces as `cflux.out`/`mch4.out`/`ngases.out`, just renamed + per-rank-appended + gzipped).
